# TASK-004: Tema claro para mejorar visibilidad

- **Proyecto:** task-manager-gui
- **Prioridad:** media
- **Creado:** 2026-03-17
- **Repo:** https://github.com/juanalvarezamaro/task-manager-gui

## Petición original
Juan ha notado que la interfaz se ve muy oscura, excepto la parte del tablón de tareas y las tarjetas de proyectos. Necesitamos un tema más claro y coherente.

## Especificaciones funcionales (Jimmy)

1. **Diseño General:**
   - Utilizar un tema claro como base
   - Mantener contraste adecuado para legibilidad
   - Asegurar coherencia visual en toda la aplicación

2. **Elementos a modificar:**
   - Fondo general de la aplicación
   - Barras de navegación/menús
   - Headers y footers
   - Sidebars o paneles laterales
   - Mantener el diseño actual del tablón Kanban y tarjetas que ya funciona bien

3. **Paleta de colores:**
   - Fondo principal: blanco o gris muy claro
   - Elementos de UI: grises claros y medios
   - Acentos: mantener los colores actuales de estados/etiquetas
   - Texto: asegurar buen contraste (negro sobre fondos claros)

4. **Consideraciones:**
   - Mantener la jerarquía visual actual
   - No sacrificar funcionalidad por estética
   - Asegurar que los estados de las tareas siguen siendo fácilmente distinguibles
   - Considerar accesibilidad (contraste, legibilidad)
